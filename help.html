GitHub의 사용방법에 대한 설명입니다.

가입 및 데스크탑 사용

GitHub의 가입, 데스크탑 다운로드 및 저장소의 파일을 다운받는 방법입니다.

GitHub 가입하기

GitHub에 가입하실 때 사용하시는 메일 주소는 핸드폰으로 확인할 수 있는 이메일 계정을 사용하시는 것이 좋고, 관리자에게 권한 요청을 하실때에는 username을 알려 주셔야 합니다.

GitHub Desktop

데스크탑 소프트웨어는 저장소의 소스 파일을 내 컴퓨터로 다운받고, Branch를 생성하거나, 수정된 소스 파일을 업로드할 수 있도록 도와줍니다.

저장소 복사

데스크탑 소프트웨어를 이용하여 저장소를 복사하고, 파일을 수정한 후에 업로드하는 방법입니다.
