도움말

GitHub의 사용방법에 대한 설명입니다.

깃허브(GitHub) 사용하기

GitHub에 가입하고, 저장소 권한을 요청하는 방법입니다.

GitHub 가입하기

GitHub에 가입하실 때 사용하시는 메일 주소는 핸드폰으로 확인할 수 있는 이메일 계정을 사용하시는 것이 좋고, 관리자에게 권한 요청을 하실때에는 username을 알려 주셔야 합니다.

프로필 사진 수정하기

프로필의 사진을 본인 사진으로 변경하여 주시기 바랍니다. 얼굴 사진을 올려 주시면, 관리자가 사람찾기가 쉽습니다.

저장소 권한 요청하기

참여하고 싶은 프로젝트를 선택하고, 저장소의 권한을 요청합니다.

코드(Code) 사용하기

코드의 파일을 본인의 컴퓨터와 동기화시키는 방법입니다.

데스크탑(Desktop)

데스크탑 소프트웨어를 내 컴퓨터에 설치하는 방법입니다.

코드 복사, 수정 및 업로드

코드 파일을 내 컴퓨터에 복사하고 서버의 저장소와 동기화하는 방법입니다.

Branch 생성하기

본인의 Branch를 생성하는 방법입니다.

Pull Request

본인의 Branch를 master와 합치는 방법입니다.

이슈(Issue) 사용하기

이슈를 올리고, 속성을 설정하는 방법입니다.

이슈 올리기

이슈를 올리면서, 첨부파일을 같이 올리는 방법입니다.

이슈에 속성 설정하기

이슈에 업무 할당, 프로젝트 설정, 플래그 설정등을 하는 방법입니다.

프로젝트(Project) 사용하기

프로젝트를 이용하여 이슈와 업무를 관리하는 방법입니다.

프로젝트 생성, 노트 생성 및 관리

프로젝트를 생성하고 노트를 추가하는 방법입니다.

이슈를 프로젝트에 연결

이슈를 프로젝트에 연결하고, 관리하는 방법입니다.

위키(Wiki) 사용하기

위키를 이용하여 설명서 혹은 자료를 관리하는 방법입니다.

위키 페이지 생성하기

위키 페이지를 작성하고 저장하는 방법입니다.
