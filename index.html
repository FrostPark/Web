GitHub Desktop 설치하기

GitHub Desktop은 내 컴퓨터와 서버 저장소를 동기화 시켜 줍니다. 설치하면 아주 손쉽게 저장소의 파일을 내 컴퓨터로 가져올 수 있으며, 수정한 내용을 저장소로 올릴 수 있습니다. Mac과 Windows 모두 지원합니다.
